# XML Parser

## Overview
**XML Parser** is a console-based Java application for reading, editing, and creating XML files without using external libraries.  
The project demonstrates a strong understanding of object-oriented programming, SOLID principles, and design patterns.


## Tech Stack
- Java
- Object-Oriented Programming (OOP)
- SOLID Principles
- Design Patterns:
  - Factory Method
  - Singleton

## Features

### XML Processing
- Read and parse XML files  
- Create new XML structures  
- Edit existing XML content  

### Data Manipulation
- Add, edit, and remove elements  
- Manage attributes  
- Modify hierarchical structure  

### Query Support
- Execute basic XPath queries  
- Search elements by criteria  

## Architecture & Design
- Fully object-oriented design  
- Clear separation of responsibilities  
- Scalable and maintainable structure  

## Key Functionalities
- XML parsing without external libraries  
- Custom implementation of XML structure handling  
- Efficient element and attribute processing
